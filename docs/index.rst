Welcome to CESSDA Metadata Aggregator's documentation!
======================================================

This documentation describes operation, usage and development of
CESSDA Metadata Aggregator.

CESSDA Metadata Aggregator is a service which exposes metadata records
gathered from OAI-PMH endpoints for re-harvesting. The project
consists of two server applications, a client and a database.

The application releases described in this documentation

* DocStore |cdcagg_docstore_version|
* Client |cdcagg_client_version|
* OAI-PMH Repo Handler |cdcagg_oai_version|
* Shared-library |cdcagg_common_version|
